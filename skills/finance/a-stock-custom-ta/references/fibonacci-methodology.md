# 斐波那契多周期分析 — 方法论文档

## 适用场景

用户明确要求"斐波那契/斐波纳切分析"，或要求"长中短三周期技术分析"时使用。

## 核心思路

取三个时间维度的价格数据，分别识别周期内的摆动高点和低点（最高价/最低价），计算斐波那契回调位和扩展位，判断当前价格在各级别中的位置。

## 数据周期选择

| 周期 | 数据跨度 | K线粒度 | 用途 |
|------|---------|---------|------|
| 长期 | 5y | 1wk (周线) | 识别主要牛熊分界，大周期支撑阻力 |
| 中期 | 1y | 1d (日线) | 判断当前趋势方向，确认关键位 |
| 短期 | 3mo | 1h (小时线) | 寻找精确进场/出场点 |

## 算法步骤

### 1. 确定摆动点

```
swing_high = period['High'].max()
swing_low  = period['Low'].min()
```

也可用更精细的方法 — 寻找最近的明显高低点（如使用zigzag算法或峰谷检测），但简单起见全周期极值已经足够。

### 2. 计算回调位

diff = swing_high - swing_low

| 水平 | 计算 | 含义 |
|------|------|------|
| 23.6% | high - diff × 0.236 | 浅回调，强势调整 |
| 38.2% | high - diff × 0.382 | **牛熊分水岭**，最关键的回调位 |
| 50.0% | high - diff × 0.500 | 心理关口，非严格Fib但市场常用 |
| 61.8% | high - diff × 0.618 | 黄金分割位，深度回调 |
| 78.6% | high - diff × 0.786 | 极深回调，危险信号 |

### 3. 计算扩展位（向上突破目标）

| 水平 | 计算 | 含义 |
|------|------|------|
| 127.2% | high + diff × 1.272 | 第一目标 |
| 138.2% | high + diff × 1.382 | 中等目标 |
| 161.8% | high + diff × 1.618 | 黄金扩展位 |
| 200.0% | high + diff × 2.000 | 双倍目标 |
| 261.8% | high + diff × 2.618 | 极端目标 |

### 4. 位置判断

```
if price >= 38.2% retracement:
    突破牛熊线，偏向多头
elif price >= 23.6%:
    反弹中，等待确认
elif price between 38.2% and 50%:
    震荡区间，观望
elif price < 50%:
    弱势，可能继续下探61.8%
```

## 辅助指标

在每个时间维度上同时输出：

- **均线 (MA20/MA50)**: 价格在均线上/下方 → 判断短期/中期趋势
- **RSI(14)**: 超买(≥70) / 超卖(≤30) / 中性
- **量比**: 当前成交量 / 20日均量 → 判断信号可靠性
- **放量/缩量**: 量比 > 1.5 放量, < 0.7 缩量

## 多周期综合研判规则

1. **长周期定方向**: 价格在长期Fib 38.2%之上还是之下决定了多空基调
2. **中周期定节奏**: 中期趋势是否配合长周期方向
3. **短周期定时机**: 用短期Fib找精确位，配合RSI判断超买超卖

## 信号强度

| 条件 | 多头信号 | 空头信号 |
|------|---------|---------|
| 价格在38.2%之上 + MA多头排列 + RSI中性 | 强多 | — |
| 价格突破前高 + 放量 | 突破确认 | — |
| 价格在38.2%下方 + MA空头排列 | — | 强空 |
| 短期RSI超买 + 价格在23.6%阻力位 | 回调风险 | 短期做空机会 |

## 输出格式

以三周期为分节，每节包含：
1. 周期高低点及日期
2. 所有回调位+当前价格相对于该位的位置
3. 扩展位
4. 均线+RSI+成交量辅助判断
5. 短线信号小结

最终输出多周期综合研判段。
