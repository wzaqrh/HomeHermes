# 静态A股浅度分析技能
当实时股票分析工具超时或网络受限无法获取实时数据时，使用本流程生成浅度分析报告。

## 执行步骤
1.  **确认目标标的**：获取A股股票代码（如600276.SS、002292.SZ）
2.  **基本面整理**：基于公开离线资料提取核心信息
   - 主营业务与行业定位
   - 核心产品/业务板块/IP矩阵
   - 行业地位与竞争格局
   - 阶段性业绩表现与转型方向
3.  **技术面静态推演**：结合公开历史数据完成分析
   - 板块定位与估值特征参考
   - 历史波动规律与关键价位参考（可选斐波那契测算）
4.  **报告生成**：输出Markdown格式分析文档
5.  **文件保存**：将报告保存至`~/feushu/`目录，命名格式为`{ticker}-static-shallow-analysis.md`
6.  **添加免责声明**：明确标注报告基于离线公开资料生成，不构成投资建议

## 适用场景
- 实时`tradingagents-runner`分析超时/失败
- 网络访问受限（无法访问东方财富/百度/实时行情接口）
- 仅需要快速浅度分析的场景

## 注意事项
- 不包含实时行情、最新财报、实时新闻数据
- 仅作参考，实际投资决策需结合最新公开信息

- 002292.SZ（奥飞娱乐）静态分析报告
- 603207静态斐波那契分析报告
