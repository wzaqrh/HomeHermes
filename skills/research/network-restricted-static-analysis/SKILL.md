# 网络受限环境下的静态分析流程

## 适用场景
1. 无法访问目标平台（东方财富、百度、OpenAI API等）返回403/超时错误
2. 需要快速生成基于公开离线资料的分析报告

## 执行步骤
1. 调用本地公开知识库/离线资料生成分析内容
2. 将结果保存至`~/feushu/`目录，命名规则：`[主题]-分析.md`
3. 注明内容基于公开离线资料，不保证实时性

## 示例
- 无人机产业A股股票清单：整理产业链上下游上市公司分类
- 伊朗革命卫队权力结构：基于公开权威报道整理核心实权派
- 地缘政治博弈分析：基于历史事件和公开资料展开维度拆解

## 注意事项
1. 禁止尝试绕过网络限制的操作
2. 所有文件仅保存至`~/feushu/`目录
3. 明确告知用户内容为静态分析，非实时数据
4. 对于需要登录的平台（如微信公众号、B站弹幕发送等），无法直接完成交互时，应直接告知用户限制并提供静态分析替代方案
